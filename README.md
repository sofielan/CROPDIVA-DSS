# CROPDIVA Decision Support System

### Retrieval-Augmented Knowledge System for Orphan Crop Research
This repository contains the document processing and embedding pipeline used in the CROPDIVA Decision Support System (DSS).
The system enables semantic search and question answering over project documents such as reports, scientific papers, and practice abstracts related to orphan crops.
The code prepares a document corpus for Retrieval-Augmented Generation (RAG) by extracting text from PDFs, splitting the text into chunks, and generating semantic embeddings.

These embeddings allow the DSS chatbot to retrieve the most relevant document fragments when answering user queries.

# System Architecture
The DSS uses 
* document embeddings
* semantic similarity search
* a large language model for answer generation
